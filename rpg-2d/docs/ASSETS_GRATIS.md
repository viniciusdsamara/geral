# 🎨 Arte, áudio e assets GRÁTIS (e legais) para o jogo

Você **não precisa gastar nada** para fazer um RPG 2D bonito. Aqui estão as
melhores fontes gratuitas — todas permitem uso em jogos (muitas até comercial).

> ⚠️ **Sempre confira a licença** de cada asset. As mais comuns:
> - **CC0** = domínio público, faça o que quiser, sem crédito obrigatório (o ideal).
> - **CC-BY** = pode usar, mas precisa **dar crédito** ao autor.
> - Guarde os créditos num arquivo `CREDITS.md` desde o começo!

## 🤖 Como eu (Claude) posso ajudar com assets

Eu trabalho com **código e arquivos**, então posso:

- ✅ Escrever scripts (GDScript/Python) para **importar e organizar** packs em massa
- ✅ Gerar **arte procedural / shaders** dentro da Godot via código
- ✅ Montar TileMaps, configurar colisão, animações por código
- ✅ Criar um `CREDITS.md` e manter a organização das licenças
- ❌ Não desenho sprites à mão nem "vejo" se a arte ficou bonita — aí **você é os olhos**

> Quando baixar um pack, é só jogar os arquivos em `assets/` e me chamar pra
> configurar o resto (TileSet, AnimatedSprite2D, etc.).
