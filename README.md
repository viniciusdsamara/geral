# geral — Sistema de gestão de obra

Meu sistema pessoal para tocar obras com método e construir track record.

## Conteúdo
- **`casa-ma/rdo-template.md`** — modelo de Relatório Diário de Obra (RDO) da Casa M&A.
- **`casa-ma/rdo/`** — os RDOs do dia a dia (um arquivo por data).
- **`licoes-aprendidas.md`** — registro de lições aprendidas (RL, Casa M&A...) para
  melhorar a próxima obra e apresentar à empresa.
